# Headless Portfolio CMS

## Updated Production Architecture Documentation

---

# Project Overview

## Plugin Name

**Headless Portfolio CMS**

## Purpose

A modern API-first portfolio CMS plugin for WordPress that allows developers, freelancers, agencies, and creators to manage structured portfolio data and expose it through clean REST APIs for modern frontend frameworks like:

* Next.js
* React
* Astro
* Nuxt
* Gatsby

---

# Plugin Vision

This plugin should function as:

> A complete headless developer portfolio content platform powered by WordPress.

Not merely:

* a portfolio showcase plugin,
* a theme addon,
* or a frontend template system.

---

# CMS Architecture

Your plugin currently includes these core entities:

* Projects
* Experience
* Education
* Resume
* Skills
* Testimonials

These should become the foundation of the CMS architecture.

# Security Requirements

Critical for WordPress.org approval.

---

# Sanitize Inputs

```php id="07d2t0"
sanitize_text_field()
sanitize_email()
esc_url_raw()
absint()
```

---

# Escape Outputs

```php id="c14dlz"
esc_html()
esc_attr()
esc_url()
wp_kses_post()
```

---

# Nonces

```php id="r8ph1f"
wp_verify_nonce()
```

---

# Permission Checks

```php id="j8lg9h"
current_user_can()
```

---

# NEVER Use

```php id="v2p2vh"
'permission_callback' => '__return_true'
```

for sensitive endpoints.
