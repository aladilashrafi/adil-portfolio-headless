API-first headless portfolio CMS for WordPress. Powers modern frontends (Next.js, React, Astro, Nuxt, Gatsby) via a clean REST API.

== Description ==

A complete headless developer portfolio content platform powered by WordPress. Manage structured portfolio data and expose it through clean REST APIs for modern frontend frameworks.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Portfolio CMS' in the menu to configure your settings.
